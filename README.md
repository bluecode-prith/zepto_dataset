# 🛒 Zepto Product Analysis using PostgreSQL

## 📌 Project Overview

This project performs business-oriented data analysis on a Zepto product dataset using PostgreSQL. The dataset was imported from an Excel/CSV file into a PostgreSQL database, followed by SQL-based analysis to generate meaningful business insights related to pricing, inventory, discounts, and revenue.

The project demonstrates practical SQL skills, including data definition, aggregation, filtering, sorting, grouping, and business analytics.

---

## 📂 Dataset

The dataset contains product information such as:

- SKU ID
- Product Name
- Category
- Maximum Retail Price (MRP)
- Discount Percentage
- Discounted Selling Price
- Available Quantity
- Product Weight (grams)
- Stock Availability
- Quantity

## 📊 Business Problems Solved

This project answers several business questions, including:

1. Top 10 products with the highest discount percentage.
2. Premium products currently out of stock.
3. Estimated revenue generated by each product category.
4. Top 10 most expensive products.
5. Products with the highest inventory value.
6. Products offering maximum customer savings.
7. Products with the highest available inventory.
8. Categories providing the highest average discount.
9. Best value products based on price per gram.
10. Categories with the highest number of out-of-stock products.
